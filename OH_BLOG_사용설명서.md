# 🚀 OH BLOG(오 블로그) 분석기 사용 설명서

**OH BLOG**는 네이버 블로그 포스팅의 SEO(검색 최적화) 상태를 분석하고, 키워드 순위 및 경쟁사를 한눈에 파악할 수 있도록 돕는 스마트한 분석 도구입니다. 이 프로그램은 별도의 설치 과정 없이 실행파일만으로 간편하게 사용할 수 있습니다.

---

## 1. 프로그램 설치 및 실행 방법

사용 중인 운영체제에 맞는 파일을 다운로드하여 실행하세요. **(Node.js 설치가 전혀 필요 없습니다.)**

### 🪟 Windows (윈도우)
1. **파일:** `OH BLOG Setup 1.0.0.exe`
2. **실행:** 파일을 더블 클릭하여 설치를 진행합니다.
3. **보안 경고 발생 시:** "Windows의 PC 보호" 창이 뜨면 **[추가 정보]**를 클릭한 후 **[실행]** 버튼을 누르세요. (인증되지 않은 개발자 경고이며 프로그램에는 문제가 없습니다.)

### 🍎 Mac (맥)
1. **파일:** 
   - Apple Silicon (M1, M2, M3 등): `OH BLOG-1.0.0-arm64.dmg`
   - Intel Chip: `OH BLOG-1.0.0.dmg`
2. **설치:** `.dmg` 파일을 열고 **OH BLOG** 아이콘을 **Applications(응용 프로그램)** 폴더로 드래그합니다.
3. **보안 경고 발생 시:** 
   - 앱 실행 시 "확인되지 않은 개발자" 메시지가 뜨면 [취소]를 누릅니다.
   - `시스템 설정` > `개인정보 보호 및 보안` > `보안` 탭으로 이동합니다.
   - 하단의 **'확인 없이 열기'** 버튼을 클릭하여 실행하세요.

## 2. 주요 기능 안내

### 🔍 포스팅 SEO 분석
- **사용법:** 분석하고 싶은 블로그 글 주소(URL)와 타겟 키워드를 입력하고 [분석하기]를 클릭합니다.
- **결과:** 100점 만점 기준의 SEO 점수, 글자 수 상세 통계, 이미지 개수 및 원본 여부를 확인할 수 있습니다.

### 🏆 키워드 순위 체크
- **사용법:** 내 블로그 주소와 키워드를 입력하여 현재 네이버 블로그 검색 결과 상위 10위 안에 내 글이 있는지 확인합니다.

### ⚔️ 경쟁사 분석
- **사용법:** 특정 키워드로 검색했을 때 상위 노출된 글들의 평균 데이터(글자 수, 이미지 수)를 내 글과 비교하여 보완점을 찾습니다.

## 3. 자주 묻는 질문 (FAQ)

**Q: 프로그램을 실행하려면 Node.js를 설치해야 하나요?**  
A: 아니요. 이 프로그램은 필요한 모든 엔진을 포함하고 있어, 일반 프로그램처럼 바로 실행하면 됩니다.

**Q: 데이터는 어디에 저장되나요?**  
A: 모든 분석 기록과 설정은 사용자의 컴퓨터 내부(`blogpulse.db`)에 안전하게 저장됩니다. 외부 서버로 데이터가 전송되지 않습니다.

**Q: 분석 결과가 정확하지 않은 것 같아요.**  
A: 네이버 블로그의 레이아웃 변경에 따라 일시적으로 분석이 어려울 수 있습니다. 이럴 경우 프로그램 업데이트를 확인해 주세요.

## 4. 고득점을 위한 SEO 팁
1. **키워드 배치:** 제목의 가장 앞부분에 핵심 키워드를 넣으세요.
2. **적정 분량:** 공백 제외 1,500자 이상의 정보성 글이 유리합니다.
3. **이미지 활용:** 직접 찍은 사진을 12장 이상 포함하고, 사진 속에 글자를 너무 많이 넣지 마세요.
4. **가독성:** 적절한 문단 나눔과 불렛포인트(•) 사용은 가산점 요인입니다.
